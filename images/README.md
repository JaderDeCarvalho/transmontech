# Technology logos (PNG)

Add these PNG files in this folder for the Technologies section:

| File           | Technology  | Where to get a PNG logo |
|----------------|-------------|--------------------------|
| `csharp.png`   | .NET & C#   | [Microsoft](https://docs.microsoft.com/en-us/dotnet), or search "C# logo PNG" |
| `angular.png`  | Angular     | [Angular](https://angular.io), or search "Angular logo PNG" |
| `sqlserver.png`| SQL Server  | [Microsoft SQL Server](https://www.microsoft.com/sql-server), or search "SQL Server logo PNG" |
| `azure.png`    | Azure       | [Microsoft Azure](https://azure.microsoft.com), or search "Azure logo PNG" |

Use square logos around 48×48 to 128×128 pixels. Name the files exactly as in the table above.
